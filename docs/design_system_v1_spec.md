# NextOwner Design System v1.0

## Brand Principles
- Premium, trustworthy, minimal.
- Designed for buying and selling businesses, not consumer ecommerce.
- Neutral-first interface with restrained color.

## Component Rules

### Buttons
Primary: blue fill, white text.
Secondary: white, gray border.
Ghost: transparent.
Destructive: red.

### Inputs
Height 48px.
Radius 12px.
Focus ring brand primary.

### Cards
White surface.
16px radius.
24px padding.
Subtle shadow.
Hover: raise shadow only.

### Navigation
Top nav white.
Active item blue.
Sidebar optional on desktop.

### Listing Cards
- Company name
- Asking price
- Revenue
- EBITDA
- Industry
- Location
- Verified badge
- Save action
- CTA

### Badges
Verified=green.
Featured=orange.
Premium=purple.
Sold=gray.

### Tables
Sticky header.
48px row height.
Alternating hover only.

### Charts
Revenue blue.
Growth green.
Costs orange.
Loss red.

## Motion
150ms hover.
250ms transitions.
350ms dialogs.

## Accessibility
- WCAG AA contrast.
- Visible keyboard focus.
- Don't rely on color alone.
- Minimum touch target 44px.

## Dark Theme
Background #020617
Surface #0F172A
Card #1E293B
Text #F8FAFC
Reuse same brand colors.

## Color Usage
70% neutrals
20% blue
10% orange
Green only for success.

## UX Principles
- One primary CTA per screen.
- Dense data, generous whitespace.
- Icons support text, never replace it.
- Prefer cards over heavy tables on mobile.
- Use confirmation for irreversible actions.
